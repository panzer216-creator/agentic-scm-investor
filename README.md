# 에이전틱 투자 시스템 (Agentic Investment System) V1.1

SCM(공급망 관리) 관점을 투자에 이식한 무인 자동화 코파일럿입니다.

## 핵심 기능
- **주도주 스캐닝:** 거래대금 폭증 상위 ETF 추출 및 Gemini API 시맨틱 필터링
- **리스크 관리:** 섹터 쿼터제(최대 3개) 및 회전율 기반 시총 편향 제거
- **동적 타점 연산:** 5일 VWAP 및 1.5 ATR 기반의 보수적 진입 타점 계산
- **기록 및 알림:** 구글 시트 장기 기억 장치 연동 및 텔레그램 작업 지시서 발송

## 실행 환경
- GitHub Actions (매일 한국 시간 오전 7시 자동 실행)
